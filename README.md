#  🔍  GitHub User Finder

A web app that lets you search for a GitHub username and view profile.

## ✨ Features
- **Search any GitHub username**
- **Display user's avatar and name**
- **Button to view full GitHub profile**

## 🚀 How It Works

**1. Type a GitHub username**

**2. Press **Enter** or click **Search****

**3. See profile picture, name, and a link to their GitHub**

## 🧰 Built With

- **HTML**
- **CSS**
- **JavaScript**
- **GitHub REST API**
